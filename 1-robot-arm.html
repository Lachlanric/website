VR Controlled Robot Arm
What did I build?

I built a 4-DoF robotic arm (excluding end effector) with a claw gripper which has the ability to be controlled using a virtual reality controller. This allows the operator to use their intuitive spacial awareness to directly adjust the end effector position in real time.

GitHub: https://github.com/Lachlanric/Robot-Arm-Snatcha/tree/main
